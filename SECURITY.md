# Security Policy

## Supported Versions

Security fixes are provided for the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report security issues by email to:

- `pratikray0709@gmail.com`

Include:

- clear reproduction steps
- impacted version or commit
- severity and impact
- proof-of-concept or logs (if available)

You will receive an acknowledgement within 72 hours.

## Disclosure Policy

- We will validate and triage the report.
- We will prepare and ship a fix.
- We will coordinate disclosure timing with the reporter.
